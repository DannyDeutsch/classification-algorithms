Assignment 3: Classification (Programming)

 

Out: 2/19/2016

Due: 3/4/2015, 11:59pm

 

Your task for this assignment is to implement and evaluate the C4.5 decision tree classifier and Naïve Bayesian classifier, and to explore SVM and other classifiers using Weka.

 

1.      Implement the C4.5 classifier and Naïve Bayesian classifier.  You are only required to handle categorical attributes but you are welcome to extend your implementation to handle numeric attributes.

a.      You can use any programming language that you are familiar with.

b.      The program should be executable with at least 3 parameters: the name of the training dataset file, the name of the test dataset file, and the name of the output file. 

c.      The program should output a file that contains the class labels for all the records in the test dataset and the classification accuracy computed as the percentage of correctly classified records in the test dataset.

2.      Evaluate your implementation using the provided mushroom dataset (mushroom.training, mushroom.test).  You will use the training set to build your classifier and the test set to evaluate its accuracy.  The provided two datasets were created using the original mushroom dataset from UCI repository, with one attribute with missing values removed. The training dataset contains 7423 records and the test dataset 701 records.  The first attribute is the class of each record and the rest 21 attributes are categorical attributes. 

3.      Explore the classifiers in Weka.

a.      Compare your implementation and results with the C4.5 and Naïve Bayesian implementation in Weka using the provided training set and test set.

b.      Explore SVM and one other classifier of your choice in Weka with at least one dataset of your choice (could be the dataset you chose for your assignment 1).

4.      Write a brief report that include the following:

a.      Present and discuss the results of your implementation on the provided dataset and other datasets (if any) and their comparison with the corresponding implementation in Weka.

b.      Present and discuss the results of your exploration using SVM and one other classifier in Weka on the dataset you selected.  For example, how do they compare in accuracy and performance?  How data preprocessing might help?

c.      Discuss, if any, the experiences and lessons you have learned from the implementation and experimentation.

5.      Your deliverable should be in one tar or zip file that contains your source files, the executable, a readme file explaining how to compile/run your program, the output file for the test dataset, and the report in pdf.

 

Note: You can work as a team of up to 2 team members.  It has to be a different team member as your assignment 2.  The grading of the implementation will be based on correctness and not on the performance.
